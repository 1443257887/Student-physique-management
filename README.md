# Student-physique-management
学生体质管理系统,JavaWeb期末答辩，JSP技术、Servlet技术
基本配置：
  JDK版本1.8
  Tomcat版本9.0
详情：
  包含用户的登录与注册（没有验证码或手机注册这些高级功能）
  学生信息的查询：
    1.精确查找某一个学生
    2.范围查找符合某条件的某些学生
  学生信息的增加：
    单个增加某学生信息，如果学号冲突则不能添加，不支持批量添加
  学生信息的更改：
    单个更改某学生的信息
  学生信息的删除：
    1.精确删除某一个学生
    2.范围删除符合某条件的一些学生
PS：
实现的功能不是很复杂，毕竟本人也不是大佬，只是个学生，有bug或不懂可以评论区交流
